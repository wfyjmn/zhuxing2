- 快速识别当前热门行业

### 3. 投资决策
- 根据行业趋势选择股票
- 对比同一行业的不同股票
- 行业轮动策略支持

## 🔧 技术实现

### 1. 数据获取
```python
# 获取行业板块信息
name_df = pro.stock_basic(
    exchange='',
    list_status='L',
    fields='ts_code,name,industry'  # 添加industry字段
)
```

### 2. 数据合并
```python
# 合并行业板块信息
df = df_a.copy()
df = df.merge(name_df, left_on='代码', right_on='ts_code', how='left')
```

### 3. 分类输出
```python
# 按行业板块分组
industries = df_output['行业板块'].unique()
for idx, industry in enumerate(industries):
    industry_df = df_output[df_output['行业板块'] == industry]
    print(industry_df.to_string(index=False))
```

## 📈 实际应用

### 场景1：板块轮动
```
📊 行业板块分布：
  新型电力: 2只  ← 近期热门
  元器件: 3只   ← 科技活跃
  化工原料: 3只 ← 周期性行业
```

**决策**：优先关注热门行业的龙头股

### 场景2：风险分散
```
如果输出10只股票：
  - 化工原料: 4只
  - 元器件: 3只
  - 新型电力: 3只
```

**决策**：选择每个行业的1-2只，分散风险

### 场景3：行业对比
```
【行业板块 1】化工原料
  股票A: 涨幅8%, PE=52
  股票B: 涨幅4%, PE=51

【行业板块 2】新型电力
  股票C: 涨幅10%, PE=33
  股票D: 涨幅6%, PE=36
```

**决策**：对比同一行业的股票，选择性价比更高的

## 🎯 使用建议

1. **关注热门板块**：股票数量较多的行业可能是资金关注的热点
2. **分散配置**：不要集中在单一行业，选择2-3个行业的股票
3. **行业对比**：对比同一行业的不同股票，选择基本面更好的
4. **板块轮动**：关注行业轮动，及时调整持仓
